Verilog Basics 1

The first set of questions would cover the fundamental knowledge of Verilog programming. These questions will help to assess the student's understanding of basic module declaration, wires interconnection, and vector concatenation.

Basics

  1. What is a port list in a Verilog module declaration? a) A list of variables declared inside the module b) A list of functions used in the module c) A list of inputs and outputs of the module d) A list of gates used in the module Answer: c) A list of inputs and outputs of the module
