class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def print_info(self):
        print(f"My name is {self.name} and I am {self.age} years old.")

person1 = Person("John", 30)
person2 = Person("Jane", 25)

person1.print_info()
person2.print_info()
  1. What is the purpose of the __init__ method in the Person class? a) To define the constructor method b) To print the person's name and age c) To create two instances of the Person class d) To define the print_info method

    Answer: a)

    Answer: c)

  1. How many instances of the Person class are created in the code snippet? a) None b) One c) Two d) Three

    Answer: c)

class Animal:
    def __init__(self, name):
        self.name = name

    def print_name(self):
        print(f"My name is {self.name}.")

class Dog(Animal):
    def __init__(self, name, breed):
        super().__init__(name)
        self.breed = breed

    def print_breed(self):
        print(f"I am a {self.breed}.")

my_dog = Dog("Fido", "Labrador")
my_dog.print_name()
my_dog.print_breed()
  1. What is the purpose of the super().__init__(name) line in the Dog class? a) To call the constructor method of the Dog class b) To initialize the name attribute of the Animal class c) To create an instance of the Animal class d) To access the name attribute of the Dog class

    Answer: b)

  1. Which class is the parent class in the inheritance relationship? a) Animal b) Dog c) Both Animal and Dog are parent classes d) There is no inheritance relationship between the two classes

    Answer: a)

  1. How many parameters does the constructor method of the Dog class have? a) None b) One c) Two d) Three

    Answer: c)

  1. Which method is specific to the Dog class and not present in the Animal class? a) __init__() b) print_name() c) print_breed() d) super()

    Answer: c)

  1. Can the Dog class access the print_name() method defined in the Animal class? a) Yes, by using the super() keyword. b) Yes, by using the self.print_name() statement. c) No, it can only access the methods defined within the Dog class. d) No, the print_name() method is not accessible in the Dog class.

    Answer: a)
