期末【測驗/USR報告】說明，不看會後悔~

問卷填了沒?   期末學習成效「學生自評」問卷

親愛的同學們：

簡單說明一下這次考試的內容，期末測驗分為兩個部分：第一部分是25道選擇題（總計50分），主要涵蓋我們最近幾週的物件導向程式設計。這部分的大部分內容是要求學生通過閱讀教材自主學習，其中有10道題目是之前已經在Zuvio（OOP 1）上公布過的，另外15道題目則是線上自學的OOP 2和OOP 3。第二部分是使用Python控制迷宮車（Maze Car），這部分的題目都是從我們在課堂上的練習中選取的，也就是說你們可以在Zuvio上找到題目。這部分的總分是80分。儘管兩個部分的成績加起來超過100分，但最終的總成績不會超過100分。

我們之前已經提到過，不論你是否參與了USR，我們都希望大家參加考試。對於參與USR的同學來說，期末成績取決於參與活動的心得報告，但通過這次考試你可以再一次驗證自己的Python學習成果。對於未參與USR的同學來說，你們應該能感受到我們在成績設計上的用心，實際上，你們所自學的部分只佔了「額外的30分」，也就是說我們是用額外分數來獎勵你的自學。根據這種計算方式，對你們的學習成果部分的評估，與參與USR的同學並沒有太大的差異，請放寬心不要太在意好像被USR的同學佔便宜了。

我只是希望大家能明白，USR是一種你們可以自由選擇的「責任義務」。我當然不希望有同學將它視為逃避的工具，無奈的是，也無法完全防止同學可能會誤用了我們的善意。然而，接受不完美也是人生成長的一部分，我希望大家能夠體會和理解這一點。

祝大家期末考順利，
