class A:
    def hello(self):
        print("Hello from A")

class B:
    def hello(self):
        print("Hello from B")

class C(A, B):
    pass

obj = C()
obj.hello()
  1. Which class does the method hello() of the object obj belong to? A) A B) B C) C D) None of the above

    Answer: A

  1. If the order of inheritance in class C was changed to class C(B, A), what would be the output of obj.hello()? A) Hello from A B) Hello from B C) It would raise an error D) None of the above

    Answer: B

  1. What is the concept demonstrated by class C inheriting from both class A and class B? A) Multiple inheritance B) Single inheritance C) Hierarchical inheritance D) Hybrid inheritance

    Answer: A

  1. If class C had its own implementation of hello() in addition to inheriting from class A and class B, which implementation would be called when calling obj.hello()? A) The implementation in class A B) The implementation in class B C) The implementation in class C D) It would raise an error

    Answer: C

from abc import ABC, abstractmethod

class Shape(ABC):
    @abstractmethod
    def area(self):
        pass

class Rectangle(Shape):
    def __init__(self, length, width):
        self.length = length
        self.width = width

    def area(self):
        return self.length * self.width

class Circle(Shape):
    def __init__(self, radius):
        self.radius = radius

    def area(self):
        return 3.14 * self.radius * self.radius

shapes = [Rectangle(5, 10), Circle(7)]

for shape in shapes:
    print(shape.area())
  1. What is the purpose of the ABC module in the code snippet? A) It stands for "Abstract Base Classes" and provides mechanisms for defining abstract classes in Python. B) It stands for "Abstract Basic Code" and is used for general code organization. C) It stands for "Abstract Base Codes" and helps in creating code snippets that are reusable. D) It stands for "Abstract Basic Concepts" and is used for representing abstract concepts in the code.

    Answer: A

  1. Which class(es) in the code snippet is/are abstract? A) Shape B) Rectangle C) Circle D) Both Shape and Circle

    Answer: A

  1. What would happen if the area() method is not implemented in the Rectangle class? A) The code would raise a TypeError. B) The code would run without any errors. C) The code would raise a NameError. D) The code would raise an AttributeError.

    Answer: A

  1. What is the purpose of the abstractmethod decorator in the Shape class? A) It indicates that the class is abstract. B) It ensures that the area() method is implemented in the derived classes. C) It allows the Shape class to be instantiated. D) It indicates that the Shape class inherits from the ABC module.

    Answer: B
